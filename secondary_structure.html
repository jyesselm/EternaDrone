Secondary Structure Layout¶

Secondary Structure is layed out in a tree structure for easy indexing of base pair and loop parameters

Secondary Structure Classes¶

ss_tree.SecondaryStructureTree([ss, seq]) Converts a dot bracket notation for secondary structure to a tree format.
ss_tree.SecondaryStructureNode Holds an element of secondary structure contained in SecondaryStructureTree. This is a base class and is never directly instantiated.

Types of Nodes¶
